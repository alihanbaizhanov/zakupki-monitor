# Система мониторинга статусов закупок

Веб-приложение для реестра закупок с ролями «Администратор» и «Руководитель» (см. [prd.md](prd.md)).

## Структура

- `server/` — Express API с общим JSON-хранилищем (единый источник данных для всех пользователей).
- `client/` — React + TypeScript + Vite фронтенд.

## Запуск

```bash
cd server && npm install && npm run dev   # http://localhost:4000
cd client && npm install && npm run dev   # http://localhost:5173
```

Фронтенд проксирует запросы `/api/*` на сервер (см. `client/vite.config.ts`).

## Возможности

- Реестр закупок с фиксированными заголовками и прокруткой.
- Редактирование ячеек по клику (только для роли «Администратор»).
- Фильтры: текстовый поиск, категория, способ закупки, статус, диапазоны дат.
- Сводная панель: всего закупок, в работе, просрочено, завершено, сумма.
- Автоматическая визуальная индикация просроченных закупок.
- Роль сохраняется в браузере пользователя, данные реестра — общие для всех.
